# Tampermonkey
A simple tampermonkey util to download PDF files on [CNKI](www.cnki.net).

## 功能说明

- 可以直接下载中国知网博硕士论文的PDF原文, 避免下载CAJ文件
- 移除了博硕士论文“分章下载”功能
- 搜索页面的“下载”按钮改为默认下载PDF
